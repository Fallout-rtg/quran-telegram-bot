# 🕌 Quran Telegram Bot

Мобильный бот для изучения Священного Корана прямо в Telegram. Предоставляет доступ к арабскому тексту, русскому переводу Э. Кулиева, толкованиям (тафсиру) шейха ас-Саади и профессиональному аудио-чтению.

**Бот работает на**: [Telegram (@quran_telegram_fallout_bot)](https://t.me/quran_telegram_fallout_bot)  
**Веб-приложение**: https://quran-telegram-bot-eight.vercel.app

---

## 🌟 Особенности

### Для Telegram-бота
- 📖 **Просмотр аятов** — запросите отдельный аят или диапазон (макс. 10 аятов)
- 🗣️ **Арабский текст** — оригинальное начертание (Усманского письма)
- 🌍 **Русский перевод** — перевод смыслов Э. Кулиева
- 📚 **Толкование** — полные пояснения шейха ас-Саади на каждый аят
- 🎵 **Аудио-чтение** — профессиональное чтение Мишари аль-Афаси
- 📜 **Список сур** — полный каталог всех 114 сур с количеством аятов
- ⌨️ **Удобная навигация** — кнопки для переключения между аятами

### Веб-приложение
- 🎲 **Случайный аят** — откройте рандомный аят каждый день
- 🔍 **Поиск по сурам** — выберите суру и конкретные аяты
- 📱 **Мобильная оптимизация** — удобен на смартфоне
- 🎧 **Плеер диапазона** — слушайте несколько аятов подряд
- 🌈 **Красивый интерфейс** — современный дизайн с поддержкой темпа

---

## 🚀 Использование

### В Telegram

#### Просмотр одного аята
```
2:255
```
Показывает аят 255 из суры 2 (Аль-Бакара) с переводом, толкованием и аудио.

#### Просмотр диапазона аятов
```
2:1-3
```
Показывает аяты 1-3 из суры 2 (максимум 10 аятов одновременно).

#### Команды
- `/start` — приветственное сообщение с инструкциями
- `/surahs` — показать полный список всех 114 сур

#### Навигация
Используйте кнопки «⬅️ Предыдущие» и «Следующие ➡️» для переключения между аятами в одной суре.

## ⚙️ Установка и развертывание

### Требования
- Python 3.7+
- Telegram Bot Token (получить от [@BotFather](https://t.me/BotFather))

### Локальное тестирование

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/Fallout-rtg/quran-telegram-bot
cd quran-telegram-bot
```

2. **Установите зависимости**
```bash
pip install -r requirements.txt
```

3. **Установите переменные окружения**
```bash
export BOT_TOKEN="your_telegram_bot_token_here"
```

4. **Запустите Flask приложение локально**
```bash
python -m flask --app api/index run
```

5. **Настройте Telegram webhook**

Для локального тестирования используйте инструменты как `ngrok`:
```bash
ngrok http 5000
```
Затем установите webhook в Telegram:
```bash
curl -X POST https://api.telegram.org/bot{BOT_TOKEN}/setWebhook \
  -d url=https://your-ngrok-url/bot-webhook
```

### Развертывание на Vercel

1. **Подключите GitHub репозиторий**
   - Перейдите на [vercel.com](https://vercel.com)
   - Импортируйте этот репозиторий

2. **Установите переменную окружения**
   - В настройках проекта установите `BOT_TOKEN`

3. **Настройте webhook в Telegram**
```bash
curl -X POST https://api.telegram.org/bot{BOT_TOKEN}/setWebhook \
  -d url=https://your-vercel-url/bot-webhook
```

Vercel автоматически развернет приложение на каждый push в main.

## 🔧 API Endpoints

### POST `/bot-webhook`
Получает обновления от Telegram и обрабатывает команды пользователя.

**Поддерживаемые форматы запроса:**
- `2:255` — один аят
- `2:1-5` — диапазон аятов
- `/start` — приветствие
- `/surahs` — список сур

### GET `/`
Простая проверка что бот работает. Возвращает `"Bot is running"`.

## 🔐 Безопасность

- **BOT_TOKEN** хранится только в переменных окружения Vercel (не коммитится)
- **CORS** не требуется (бот общается только с Telegram API)
- **Валидация** входных данных перед запросом к API
- **Timeout** для всех HTTP запросов (10 секунд)
